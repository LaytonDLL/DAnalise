Tableau como Alternativa ao Power BI

Conheca o Tableau, uma das ferramentas de BI mais poderosas do mercado, e entenda quando usa-lo.

O que e Tableau?

Tableau e uma plataforma de Business Intelligence focada em visualizacao de dados. Adquirida pela Salesforce em 2019, e conhecida por sua capacidade de criar visualizacoes complexas de forma intuitiva atraves de drag-and-drop.

[+] Pontos Fortes do Tableau

  • Visualizacoes avancadas e altamente personalizaveis
  • Melhor para exploracao de dados (analytics)
  • Comunidade forte com muitos recursos gratuitos
  • Tableau Public gratuito para portfolios
  • Preferido por empresas de tecnologia e startups

Tableau vs Power BI

Aspecto Tableau Power BI
Preco Mais caro (~$70/usuario/mes) Mais barato (~$10/usuario/mes)
Visualizacoes Mais flexivel e avancado Bom, mas mais limitado
ETL Tableau Prep (separado) Power Query (integrado)
Linguagem Calculated Fields (mais simples) DAX (mais poderoso)
Integracao Melhor com Salesforce Melhor com Microsoft
Curva de Aprendizado Mais intuitivo no inicio Mais complexo, mas mais recursos
Mercado Brasil Menos comum Mais comum (80%+)
Versao Gratuita Tableau Public (publica) Power BI Desktop (local)

[*] Recomendacao de Carreira

No Brasil, Power BI e mais requisitado (maioria das vagas). Porem, conhecer Tableau amplia suas opcoes, especialmente em empresas internacionais e startups. O ideal e dominar um e conhecer o outro.

Produtos Tableau

Tableau Desktop

Aplicacao principal para criar dashboards. Equivalente ao Power BI Desktop.

Tableau Server/Cloud

Plataforma para publicar e compartilhar. Equivalente ao Power BI Service.

Tableau Prep

Ferramenta de ETL visual. Equivalente ao Power Query.

Tableau Public

Versao gratuita mas publica. Otimo para portfolio!

Conceitos Basicos do Tableau

Dimensoes vs Medidas

Dimensoes (Azul)

Campos categoricos que segmentam os dados. Ex: Produto, Regiao, Data.

Medidas (Verde)

Campos numericos que sao agregados. Ex: Vendas, Quantidade, Lucro.

Shelves (Prateleiras)

Shelf Funcao Equivalente Power BI
Columns Define eixo X Eixo X
Rows Define eixo Y Eixo Y
Color Colore por categoria Legenda
Size Dimensiona por valor Tamanho
Filters Filtra dados Filtros
Pages Cria animacao/paginacao Sem equivalente direto
Marks Tipo de visualizacao Tipo de grafico

Calculated Fields

Equivalente a medidas DAX, mas com sintaxe mais simples:

// Calculo basico [Vendas] - [Custo] // IF/THEN IF [Lucro] > 0 THEN "Positivo" ELSE "Negativo" END // CASE CASE [Regiao] WHEN "Norte" THEN "N" WHEN "Sul" THEN "S" ELSE "Outros" END // Agregacoes SUM([Vendas]) AVG([Preco]) COUNTD([Cliente ID]) // Table Calculations (Window Functions) RUNNING_SUM(SUM([Vendas])) RANK_UNIQUE(SUM([Vendas]), 'desc') ZN(SUM([Vendas])) - LOOKUP(ZN(SUM([Vendas])), -1) // Diferenca vs anterior

[*] LOD Expressions

Level of Detail Expressions sao um recurso poderoso e unico do Tableau. Permitem calculos em diferentes niveis de granularidade.

// Vendas por cliente (independente de filtros) { FIXED [Cliente ID] : SUM([Vendas]) } // Media de vendas por regiao { INCLUDE [Regiao] : AVG([Vendas]) }

Criando seu Primeiro Dashboard

  • Conectar Dados: Data > Connect to Data Source
  • Explorar: Arraste campos para Rows e Columns
  • Escolher Visualizacao: Show Me panel ou Marks card
  • Formatar: Cores, rotulos, titulos
  • Criar Sheets: Uma visualizacao por sheet
  • Montar Dashboard: New Dashboard > arraste sheets
  • Adicionar Interatividade: Filters, Actions, Highlights
  • Publicar: Server > Publish Workbook

Tableau Public - Portfolio Gratuito

Tableau Public e a versao gratuita onde voce pode criar e publicar visualizacoes. E excelente para construir portfolio!

[+] Vantagens do Tableau Public

  • 100% gratuito para sempre
  • Publique visualizacoes online
  • Compartilhe link do seu perfil
  • Explore visualizacoes de outros usuarios
  • Recrutadores podem ver seu trabalho

[!] Limitacoes

  • Dados ficam publicos (nao use dados sensiveis)
  • Sem conexao a databases (so arquivos)
  • Limite de 10 milhoes de linhas
