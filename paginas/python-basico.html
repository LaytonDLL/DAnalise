Dominar Python do Basico a Analise de Dados

Aprenda Python do zero ate criar analises profissionais com as bibliotecas mais usadas do mercado.

Por que Python para Analise de Dados?

Python e a linguagem mais popular para analise de dados, ciencia de dados e machine learning. Sua sintaxe simples e ecossistema rico de bibliotecas a tornam ideal para iniciantes e profissionais.

[+] Vantagens do Python

  • Linguagem #1 em Data Science e Machine Learning
  • Sintaxe limpa e facil de aprender
  • Bibliotecas poderosas (Pandas, NumPy, Scikit-learn)
  • Grande comunidade e muitos recursos gratuitos
  • Requisito em 70%+ das vagas de analista senior
  • 100% gratuito e open source

Configurando o Ambiente

Instalacao Recomendada

  • Anaconda: Distribuicao Python com todas bibliotecas de dados pre-instaladas
  • VS Code: Editor de codigo leve e poderoso (extensao Python)
  • Jupyter Notebook: Ambiente interativo para analise (vem com Anaconda)
  • Google Colab: Alternativa online gratuita, sem instalacao

Fundamentos de Python

Variaveis e Tipos de Dados

# Tipos basicos nome = "Maria" # string (texto) idade = 25 # int (inteiro) salario = 5500.50 # float (decimal) ativo = True # bool (booleano) # Colecoes lista = [1, 2, 3, 4, 5] # lista (mutavel) tupla = (1, 2, 3) # tupla (imutavel) dicionario = {"nome": "Maria", "idade": 25} # dict (chave-valor) conjunto = {1, 2, 3} # set (valores unicos)

Estruturas de Controle

# Condicional if idade >= 18: print("Maior de idade") elif idade >= 16: print("Pode votar") else: print("Menor de idade") # Loop for for i in range(5): print(i) # Loop com lista for item in lista: print(item) # List comprehension (Pythonico!) quadrados = [x**2 for x in range(10)]

Funcoes

Jupyter Notebook

O Jupyter Notebook e o ambiente padrao para analise de dados. Permite combinar codigo, visualizacoes e texto em um unico documento.

Atalhos Essenciais

  • Shift + Enter: Executar celula
  • B: Nova celula abaixo
  • A: Nova celula acima
  • DD: Deletar celula
  • M: Celula Markdown
  • Y: Celula codigo

Boas Praticas

  • Use Markdown para documentar
  • Execute celulas em ordem
  • Reinicie kernel periodicamente
  • Exporte como HTML para compartilhar
