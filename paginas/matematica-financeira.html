Juros Simples

No regime de juros simples, os juros sao calculados sempre sobre o valor inicial (capital).

J = C x i x t
M = C + J = C x (1 + i x t)

Onde: J = Juros | C = Capital | i = Taxa | t = Tempo | M = Montante

Exemplo

Capital: R$ 10.000 | Taxa: 2% ao mes | Tempo: 6 meses

J = 10.000 x 0,02 x 6 = R$ 1.200
M = 10.000 + 1.200 = R$ 11.200

Juros Compostos

No regime de juros compostos, os juros sao calculados sobre o montante acumulado (juros sobre juros).

M = C x (1 + i)^t
J = M - C

Exemplo

Capital: R$ 10.000 | Taxa: 2% ao mes | Tempo: 6 meses

M = 10.000 x (1,02)^6 = R$ 11.261,62
J = 11.261,62 - 10.000 = R$ 1.261,62

Comparacao: Juros compostos geraram R$ 61,62 a mais que juros simples!

Porcentagem e Variacao

Variacao Percentual

Variacao % = ((Valor Final - Valor Inicial) / Valor Inicial) x 100

Exemplo: Crescimento de Vendas

Janeiro: R$ 50.000 | Fevereiro: R$ 58.000

Crescimento = ((58.000 - 50.000) / 50.000) x 100 = 16%

Aumento e Desconto

Operacao Formula Exemplo
Aumento de X% Valor x (1 + X/100) 100 + 20% = 100 x 1,20 = 120
Desconto de X% Valor x (1 - X/100) 100 - 15% = 100 x 0,85 = 85
Margem sobre custo Custo x (1 + Margem) Custo 80 + 25% = 100
Margem sobre venda Venda x Margem Venda 100 x 20% = 20 de lucro

Indicadores de Investimento

ROI - Retorno sobre Investimento

ROI = ((Ganho - Investimento) / Investimento) x 100

Exemplo: Campanha de Marketing

Investimento: R$ 5.000 | Receita gerada: R$ 15.000

ROI = ((15.000 - 5.000) / 5.000) x 100 = 200%

Para cada R$ 1 investido, retornaram R$ 2 de lucro.

Payback - Tempo de Retorno

Payback = Investimento Inicial / Fluxo de Caixa Mensal

Exemplo

Investimento: R$ 24.000 | Lucro mensal: R$ 3.000

Payback = 24.000 / 3.000 = 8 meses

Margem de Lucro

Margem Bruta = (Lucro Bruto / Receita) x 100
Margem Liquida = (Lucro Liquido / Receita) x 100

VPL e TIR

VPL - Valor Presente Liquido

Calcula o valor presente de fluxos de caixa futuros, descontados a uma taxa.

=VPL(taxa; valor1; valor2; ...) + Investimento_Inicial

Regra: VPL > 0 = Projeto viavel | VPL < 0=Projeto inviavel

TIR - Taxa Interna de Retorno

E a taxa que faz o VPL ser igual a zero. Representa a rentabilidade do projeto.

=TIR(valores; estimativa)

Regra: TIR > Taxa minima de atratividade = Projeto viavel

Exemplo Pratico

Investimento: -R$ 100.000 (ano 0)

Fluxos: R$ 30.000 (ano 1), R$ 40.000 (ano 2), R$ 50.000 (ano 3), R$ 35.000 (ano 4)

=VPL(10%; 30000; 40000; 50000; 35000) - 100000

Se VPL for positivo e TIR maior que 10%, o projeto e viavel!
